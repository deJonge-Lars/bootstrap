Dark Mode
Details

Today I wanted to create a webpage that I could turn from a light theme style into a dark theme style with a single click on a button. This is also the first time ever that I have created something like this. I did it in a trial way. If I would have made a dark theme option for my whole portfolio, I would have done it differently for sure. The thing I would have done if I made it for my whole portofolio is by creating variable styles that I could easily call on.
How I have done it now is true instead of using html['data-theme=dark'] and then the class or something else.
