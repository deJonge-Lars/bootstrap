Details

Classiczcraft was a personal project from a friend named Sergio and myself. We had created a Minecaft server to play custom minigames with some friends of us. The server started small with around 5 people to play the minigames but then friends starting inviting other friends and the server became more active. In total we had around 12 minigames that we both had created with custom maps & plugins.

The server was up for around 6 months until we both unfortunately became unactive with the server because of other games and school. On a daily base we had around 27 active people on the server to the day we closed the server. I have learned quite a bit from running the server on how to deal with the financial and support sides of the business.
