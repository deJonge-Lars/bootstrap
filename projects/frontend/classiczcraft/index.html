Welkom!

Hallo beste bezoekers,

Leuk dat jullie kijken op de site van de ClassiczCraft Server. Op de website kan je allerlei verschillende informatie vinden over de server. Als je de server echt leuk vind kun je donaten. Voor als je vragen niet beantwoord worden op de server kan je hier contact formulier invullen zodat je vraag behandeld kan worden.

Maar wacht er is nog meer!
Wil jij leuke gebouwen maken zonder dat mensen je storen dat kan op deze server! Wil je ook spelen zonder lagg en dat 24/7 lang! Maar je hebt nog veel meer!
Je kunt hier: Bouwen in freebuild, Minigames spelen, PVPen tegen andere spelers wil jij dit? Kom dan nu op deze Server! Hier zijn ook oplettende admins.

Wil je op de server?
Het enige wat JIJ moet hebben is een betaald minecraft account, crack accounts worden niet toegestaan op deze server.
